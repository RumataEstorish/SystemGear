SystemGear shows information about hardware and OS of your watch.
Categories:
* CPU (live load)
* Storage
* Display
* Platform
* Battery
* Peripheral
* WiFi
* Cellular
* Sim
* And other info

Other features:
* CPU load and battery status refreshes automatically.
